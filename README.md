# Overview

In this project, I built a REST API with authentication and role-based access (user & admin). I also created a simple frontend to test the APIs.

## What I did
User register & login (JWT + password hashing)
Role-based access (admin vs user)
CRUD APIs 
Basic validation & error handling
API testing using Postman
Simple frontend (Next) for:
login/register
protected dashboard
CRUD operations
Tech Stack

Node.js, Express, JWT, Database (MongoDB), Next.js, Bycrpt

## Run Project

Backend:

npm install
node server.js

Frontend:

npm install
npm run dev



This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).
